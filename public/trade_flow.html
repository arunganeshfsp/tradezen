⛔ HARD NO'S — Never Break These
🚫A bullish setup doesn't apply if open was below BC — CPR bias is bearish
🚫A bearish setup doesn't apply if open was above TC — CPR bias is bullish
🚫No edge if lean score < 3 — conflict day
🚫Stand aside if VIX > 25 — moves are random, not directional
🚫Skip if ORB range > 200 pts — too wide, whipsaw risk
🚫One setup per side per day — no re-engagement after invalidation
🚫Intraday study only — nothing carried overnight, flat by 3:15 PM
✅ QUICK RULES — Always Follow
📌Wait for full 5-min candle close beyond ORB level before a setup is valid
📌Invalidation = opposite side of ORB (bullish setup → invalidation at ORB Low)
📌First reference T1 (R1/S1) — partial milestone, reduces risk
📌Move invalidation to breakeven after T1 — rest rides free
📌Full conviction only on full bull/bear day + VIX < 16 + narrow CPR
📌Conditional setup → half conviction (50%) by default
📌VIX 20–25 → reduce conviction to 25% regardless of scenario
